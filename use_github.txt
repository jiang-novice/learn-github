let's start!

step 1: You need to have a GitHub account and create a repository firstly

step 2: Get the local SSH key to complete the transfer between the local 
        repository and GitHub repository
        {
            $ ssh-keygen -t rsa -C "youremail@example.com"
            
            ~/ 下生成 .ssh 文件夹，进去，打开 id_rsa.pub，复制里面的 key

            回到 github 上，进入 Account => Settings（账户配置）:
                左边选择 SSH and GPG keys，然后点击 New SSH key 按钮,title 设置标题，
                可以随便填，粘贴在你电脑上生成的 key。

            test:
            {
                $ ssh -T git@github.com

                The authenticity of host 'github.com (52.74.223.119)' can't be established.
                RSA key fingerprint is SHA256:nThbg6kXUpJWGl7E1IGOCspRomTxdCARLviKw6E5SY8.
                Are you sure you want to continue connecting (yes/no/[fingerprint])? yes                   # 输入 yes
                Warning: Permanently added 'github.com,52.74.223.119' (RSA) to the list of known hosts.
                Hi tianqixin! You've successfully authenticated, but GitHub does not provide shell access. # 成功信息

            }
        }

step 3 : Initialize local repository
{
    git init 
    ... do something you want
}

step 4 : to github
{
    $ git remote add origin git@github.com:tianqixin/runoob-git-test.git  (just first time, 
                                                                        origin: name of repository,
                                                                        git@github.com:tianqixin/runoob-git-test.git: ssh address of repository  )
    $ git push origin master  (master: local branch for push)
}
step 5: from github
{
    $ git pull origin master 
    用于从远程仓库（通常是指定为 "origin"）的 "master" 分支拉取更新并合并到当前所在的本地分支。

    
}


ps:
1. 修改现有的远程仓库地址：如果你需要修改 origin 的远程地址，可以使用 git remote set-url 命令。

    git remote set-url origin new-git-url
    其中 new-git-url 是你想要设置的新的远程仓库地址。
